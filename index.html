Projects

Here are some of the things I've built recently

North American Building Material Distribution Association

The NBMDA web redesign project entailed the complete overhaul of the client's digital presence, infusing a fresh and contemporary aesthetic to enhance the overall user experience for its members. My pivotal role as the front-end developer involved translating the creative team's designs into a meticulously crafted website. In tandem with our back-end developers, I orchestrated the integration of CRM data through APIs into an interactive application seamlessly embedded in the customer-facing website. The back-end service integration included a custom built member company map with state-by-state filtering.

  • HTML
  • SCSS
  • jQuery

Society for Interventional Oncology

A completely redesigned website for the Society for Interventional Oncology, built on DotNetNuke (DNN). As the front-end web developer on this project I collaborated with graphic designers on the creative team, project managers, and the marketing team to implement high-fidelity designs from Adobe XD into a responsive and interactive website. The design features tailored elements like a news ticker, large circular video, carousel for upcoming events, and animations using the Animate On Scroll plugin. The site is optimized for scalability, and is built using SCSS for clean and intuitive code.

  • HTML
  • SCSS
  • jQuery

Neurocritical Care Society

This project involved the implementation of a series of complex designs for the Neurocritical Care Society's suite of websites. As the front-end developer, I worked closely with the creative team and client project managers on building out NCS' main site, in addition to their Curing Coma microsite, and Currents content hub. All three sites were built on DNN with cohesive design elements to tie them together. The project required extending and developing the EasyDNN News module for several features including a custom built, searchable, filterable hub of education courses, as well as dynamic content displayed on the main site from their content hub.

  • HTML
  • CSS
  • jQuery

National Association of Orthopaedic Nurses

The website redesign for the National Association of Orthopaedic Nurses, which involved cross-team collaboration between creative directors, marketers, back-end developers, and myself as the sole front-end developer. Built on DNN, I transformed the provided Adobe XD design files from the creative team into pixel-perfect representations on the web. My work included developing and custom styling several modules: EasyDNN Rotator as an event carousel, EasyDNN News as an Education Catalog including search and filter options, as well as a separate instance used for a news blog, and the EasyDNN Simple Forum to house member discussions.

  • HTML
  • CSS
  • jQuery

Texas Speech-Language Hearing Association

In my capacity as the lead front-end developer for this project, I worked closely with our creative team to implement a refreshed and contemporary design as the client migrated their website to a new platform (DNN) and AMS. The design concept came to life through subtle animations in the hero image carousel, icons, and a "drawn" border animation on select elements all built with lightning fast SCSS and jQuery. Other features that I built out during the development include a custom mega-menu, and a simple yet distinctive event design with a separate dedicated subnavigation for those specialized pages.

  • HTML
  • CSS
  • jQuery

Turner & Emily Uligian's Wedding

In a change of pace from my typical work I undertook a much more personal endeavor - the creation of a website for my wife's and my wedding. From inception to execution, I conceived and actualized every facet of this project (with some oversight from the project boss, my wife, of course). Leveraging GitHub Pages, I employed the fullPage.js plugin and brought to life a single-page site with fluid full-page scrolling. The site features a timeline of my own design, SVG animations self-created in InDesign, a countdown clock to the big day, a user-friendly RSVP tool, and photo gallery built with the Google Photos Material Gallery plugin.
