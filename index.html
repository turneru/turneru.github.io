Projects

Here are some of the things I've built recently

Society for Interventional Oncology

A redesigned website for the Society for Interventional Oncology (SIO), built on DotNetNuke (DNN). As the front-end web developer on this project I collaborated with graphic designers on the creative team, project managers, and the marketing team to implement the final designs from Adobe XD into DNN. The design features a news ticker, large circular video, carousel for upcoming events, and animations using the Animate On Scroll plugin. The site is fully responsive, and is built using SCSS for cleaner more intuitive code.

  • HTML
  • SCSS
  • jQuery

Neurocritical Care Society

This project involved the implementation of new designs for the Neurocritical Care Society's suite of websites. As the front-end developer, I collaboarated with the creative team and project managers on building out NCS' main site, in addition to their Curing Coma microsite, and Currents content hub. All three sites were built on DNN with cohesive design elements to tie them together. Features include a custom built, searchable, filterable hub of education courses and products, a seprate event design and sub-navigation, and dynamic content displayed on the main site from their other sites.

  • HTML
  • CSS
  • jQuery

National Association of Orthopaedic Nurses

The website redesign for the National Association of Orthopaedic Nurses, which involved cross-team collaboration between myself as the lead developer with designers, marketers, and the IT manager. Built on DNN, the development included customizing several modules: EasyDNN Rotator as an event carousel, and EasyDNN News as an Education Catalog including search and filter options, as well as a separate instance used for a news blog.

  • HTML
  • CSS
  • jQuery

Texas Speech-Language Hearing Association

In my capacity as the lead front-end developer for this project, I collaborated closely with our creative team to implement a refreshed and contemporary design as the client migrated their website to a new platform (DNN) and AMS. The design concept came to life through subtle animations in the hero image carousel, icons, and a drawn border animation on select elements. Other features that were built out include a custom mega-menu and a simple event design with a separate dedicated subnavigation for those specialized pages.

  • HTML
  • SCSS
  • jQuery

Turner & Emily Uligian's Wedding

In a change of pace from my typical work I undertook a much more personal endeavor - the creation of a website for my wife Emily and I's wedding. From inception to execution, I conceived and actualized every facet of this project (with some oversight from the project boss, Emily, of course). Leveraging GitHub Pages, I employed the fullPage.js plugin and brought to life a single-page site with fluid full-page scrolling. The site features a custom designed timeline, SVG animations created in InDesign, a countdown clock to the big day, a user-friendly RSVP tool, and photo gallery built with the Google Photos Material Gallery plugin.

  • HTML
  • CSS
  • jQuery
